# 🌦️ Weather App

A simple and responsive **Weather App** built with **React + Vite** that fetches real-time weather data using a **Weather API** (e.g., OpenWeatherMap API).

---

## 🚀 Features
- 🌍 Search weather by city name
- 🌡️ Display temperature, humidity, and weather conditions
- 🖼️ Weather icons based on conditions
- 📱 Responsive design for mobile and desktop
- ⚡ Fast build and hot-reload with **Vite**

---

## 🛠️ Tech Stack
- [React](https://react.dev/) – UI library  
- [Vite](https://vitejs.dev/) – Build tool  
- [Weather API](https://openweathermap.org/api) – Real-time weather data  
- [Tailwind CSS] (optional, if you used it) – Styling

## 📂 Project Setup

### 1. Clone the repo
```bash
git clone https://github.com/your-username/weather-app.git
cd weather-app

2. Install dependencies
npm install

3. Create .env file

Inside the project root, create a .env file and add your API key:

VITE_WEATHER_API_KEY=your_api_key_here


⚠️ Never commit your API key to GitHub. Always use .env.

4. Run the app
npm run dev


Open http://localhost:5173
 in your browser.
